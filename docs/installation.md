# Installation

L'installation est décrite en détail dans le [README](../README.md#installation).

## Résumé

### HACS (recommandé)

1. HACS → trois points → `Dépôts personnalisés`
2. Dépôt : `https://github.com/foXaCe/homeassistant_veolia/` — Type : `Intégration`
3. Installez « Veolia », redémarrez Home Assistant

### Manuelle

1. Copiez `custom_components/veolia` dans le dossier `custom_components` de votre configuration
2. Redémarrez Home Assistant

## Prérequis

- Home Assistant ≥ 2025.1.0
- Un compte client actif sur un [portail Veolia supporté](../README.md#portails-veolia-supportés)
