# Security Policy

## Supported versions

Seules les versions publiées depuis la branche `main` sont supportées.

## Reporting a vulnerability

NE PAS ouvrir d'issue publique pour une faille de sécurité.

Contactez le mainteneur [@foXaCe](https://github.com/foXaCe) via un
[private vulnerability report](https://github.com/foXaCe/homeassistant_veolia/security/advisories/new).

Délai de réponse cible : 7 jours.
